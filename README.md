    "# Expedia Session-Based Hotel Recommender\n",
    "\n",
    "## 📌 Overview\n",
    "\n",
    "This project implements a session-based recommendation system using the Expedia dataset.\n",
    "The goal is to predict the next hotel a user is likely to click based on their current browsing session.\n",
    "\n",
    "The approach uses **Word2Vec embeddings** to model relationships between hotels from click sequences.\n",
    "\n",
    "---\n",
    "\n",
    "## 🧠 Problem Definition\n",
    "\n",
    "Users browse multiple hotels within a session, and their preferences can change dynamically.\n",
    "This project formulates recommendation as a **next-item prediction task** using sequential interaction data.\n",
    "\n",
    "---\n",
    "\n",
    "## 📊 Dataset\n",
    "\n",
    "* Source: https://github.com/ExpediaGroup/pkdd22-challenge-expediagroup/releases\n",
    "* Each session is a sequence of hotel IDs\n",
    "* Preprocessing steps:\n",
    "\n",
    "  * Convert click strings → ordered sequences\n",
    "  * Remove sessions with < 2 interactions\n",
    "  * Sample 5% of data for feasibility\n",
    "  * Split into:\n",
    "\n",
    "    * 80% training\n",
    "    * 10% validation\n",
    "    * 10% test\n",
    "\n",
    "---\n",
    "\n",
    "## ⚙️ Methodology\n",
    "\n",
    "### Word2Vec Modeling\n",
    "\n",
    "* Sessions → sentences\n",
    "* Hotels → words\n",
    "* Model: Skip-gram with negative sampling\n",
    "\n",
    "### Best Configuration\n",
    "\n",
    "* vector_size = 64\n",
    "* window = 3\n",
    "* negative = 8\n",
    "* alpha = 0.01\n",
    "* min_count = 5\n",
    "* epochs = 10\n",
    "\n",
    "---\n",
    "\n",
    "## 🔍 Recommendation Strategies\n",
    "\n",
    "1. **Popularity Baseline**\n",
    "\n",
    "   * Rank hotels by frequency\n",
    "\n",
    "2. **Previous-Item Model**\n",
    "\n",
    "   * Use last clicked hotel as query\n",
    "\n",
    "3. **Context-Based Model**\n",
    "\n",
    "   * Average all previous hotel embeddings\n",
    "\n",
    "---\n",
    "\n",
    "## 📏 Evaluation\n",
    "\n",
    "* Metric: **Hits@10**\n",
    "* Task: Predict final click in each session\n",
    "* Input: all previous clicks\n",
    "* Note: Sessions with missing embeddings are skipped\n",
    "\n",
    "---\n",
    "\n",
    "## 📈 Results\n",
    "\n",
    "| Model            | Hits@10 |\n",
    "| ---------------- | ------- |\n",
    "| Popular baseline | 0.0035  |\n",
    "| Previous-item    | 0.0592  |\n",
    "| Context-based    | 0.0405  |\n",
    "\n",
    "👉 Embedding-based models significantly outperform the baseline.\n",
    "👉 The **previous-item model performs best**, highlighting the importance of recent interactions.\n",
    "\n",
    "---\n",
    "\n",
    "## 💡 Key Insights\n",
    "\n",
    "* Sequential behavior improves recommendations\n",
    "* Recent interactions carry the strongest signal\n",
    "* Data sparsity remains a major challenge\n",
    "\n",
    "---\n",
    "\n",
    "## ⚠️ Limitations\n",
    "\n",
    "* Sparse data affects rare hotels\n",
    "* Word2Vec cannot model long-range dependencies\n",
    "* Some sessions are skipped due to missing embeddings\n",
    "\n",
    "---\n",
    "\n",
    "## 🚀 Future Work\n",
    "\n",
    "* Try RNNs / Transformers\n",
    "* Add hotel metadata (price, location)\n",
    "* Use additional metrics (MRR, NDCG)\n",
    "\n",
    "---\n",
    "\n",
    "## 🛠️ Tech Stack\n",
    "\n",
    "- Python  \n",
    "- Pandas, NumPy (data processing)  \n",
    "- Matplotlib, Seaborn (visualization)  \n",
    "- Scikit-learn (data splitting)  \n",
    "- Gensim (Word2Vec embeddings)\n",
